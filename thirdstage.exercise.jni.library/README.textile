
h3. Build Requirement

* JDK
** Oracle JDK 1.7 or higher 32-bit edition.

* Tools and Libraries
** *MinGW*
** *FFmpeg* 32-bit *dev* type and *shared* type  binaries
** *OpenCV* 2.3.1 distributable which including binaries for MinGW

* Environment Variables
** *@MINGW_HOME@*
** *@FFMPEG_DEV_HOME@* and *@FFMPEG_SHARED_HOME@*
** *@OPENCV_BASE@*
*** under which includes @build/x86/mingw@ directory.
