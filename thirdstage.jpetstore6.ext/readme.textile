
h1.  JPetStore Extended


* This project requires jpetstore-6 project is in the same list of Eclipse, but doesn't need the location of the jpetstore-6 is under the same parent directory with this.

* JPetStore 6 
** Source : https://github.com/mybatis/jpetstore-6
** Guide : http://mybatis.github.io/spring/sample.html

h2. Requirements or configuration

* JPetStore 6 is based on Servlet 2.5 and JSP 2.1. So, Jetty 7 is best match and there's no reason to use others.
* Necessary artifacts of Jetty are
** jetty-server for core
** jetty-jsp for javax.el
